# Finance Dashboard (Finora)

A clean and interactive finance dashboard built using React.

---

##  Features

*  Dashboard overview (Balance, Income, Expenses)
*  Charts (Balance trend, Spending breakdown)
*  Transactions table:

  * Search
  * Filter (income/expense)
  * Add / Edit / Delete (Admin)
*  Role-based UI (Admin / Viewer)
*  Insights section
*  Responsive design
*  Data persistence using localStorage

---

##  Tech Stack

* React (Vite)
* Tailwind CSS
* Recharts
* React Icons

---

##  Roles

* **Viewer** → Can only view data
* **Admin** → Can add, edit, and delete transactions

---

##  Setup

```bash
npm install
npm run dev
```

---

##  Approach

* Component-based architecture
* Utility functions for calculations and formatting
* State managed using React hooks
* LocalStorage used for persistence

---

##  Notes

This project focuses on frontend UI, UX, and state management using mock data.
